## 题型

1. 有递推关系的.
例如: dp[i] = {dp[0] ~dp[i - 1]}, 可以由之前的状态转换而来的.
例如：一维背包问题.

2. LCS
dp[i][j] = max(dp[i - 1][j], dp[i][j - 1]);
if (a[i] == b[j]) {
    dp[i][j] = max(dp[i][j], dp[i - 1][j - 1] + 1);
}

3. 概率dp
概率 = sum{所有情况的概率}, 非常重要的一点是可能会有转化为自己的情况, 这个时候，需要列公式, 求期望.

4. 博弈
① 只要是能转移到失败状态，那么当前状态就是必胜状态.
